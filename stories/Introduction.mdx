## Architecture

This library follows a **primitive-first approach** inspired by Radix UI and assistant-ui:

### Runtime Layer
Protocol-specific implementations handle communication with agents:
- `createA2ARuntime()` - A2A Protocol (JSON-RPC + SSE)
- `createACPRuntime()` - ACP Protocol (REST/HTTP) 
- `createCustomRuntime()` - Custom protocol adapters

### Component Layer
Primitive components provide building blocks for complex UIs:
- `TaskPrimitive.*` - Task creation, progress, and results
- `AgentPrimitive.*` - Agent information and capabilities

### Hook Layer
React hooks provide state management and protocol communication:
- `useAgent()`, `useTask()`, `useTaskCreation()`

## Getting Started

```tsx
import { 
  AgentProvider, 
  TaskPrimitive, 
  createA2ARuntime 
} from '@agentarea/ui-sdk'

const runtime = createA2ARuntime({
  endpoint: 'http://localhost:9999'
})

function App() {
  return (
    <AgentProvider runtime={runtime}>
      <TaskPrimitive.Root>
        <TaskPrimitive.Input onSubmit={handleTask} />
        <TaskPrimitive.Send />
        <TaskPrimitive.Progress />
        <TaskPrimitive.Output />
      </TaskPrimitive.Root>
    </AgentProvider>
  )
}
```
